# Odin Recipes

A small static HTML site built for [The Odin Project](https://www.theodinproject.com/) Foundations course. The home page links to three recipe pages; each page describes a dish with an image, description, ingredients, and steps.

## Skills demonstrated

By completing this project, you practice: valid HTML5 document structure, semantic headings and lists, links between pages with correct relative paths, embedding images, and organizing a multi-page site in folders. Styling is intentionally minimal so the focus stays on HTML structure.
